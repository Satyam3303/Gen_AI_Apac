# AI Productivity Assistant

A multi-agent AI system to manage tasks, notes, and workflows.

## Features
- Add tasks
- View tasks
- Add notes
- Plan your day

## Tech Stack
- FastAPI
- Firestore
- Cloud Run
- ADK architecture
